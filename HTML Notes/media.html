Media tag of HTML:

There are two type of media tag:
  1. Vedio tag: This tag is used to add a vedio in webpage.

    2. Attributes of the Vedio tag

    src=" ",control=" ",muted=" ", loop=" ", autoplay=" ",height=" ",width=" "
  2. Audio tag: This tag is used to add a audio in webpage.

    3. Attributes of the Audio tag

    src=" ",control=" ",muted=" ", loop=" ",autoplay=" "
