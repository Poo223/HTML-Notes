HTML INTRODUCTION:

HTML: HTML is the standard markup language for creating Web pages.
Hypertext:I t is a text where if we click on it, it will navigate us to other webpage.
Markup language:markup language, we cannot write any logic or perform any task in html.

Tags

  1. Headtag:
    • Inside the html tag there was a head tag inside head tag there was a title tag.
    • Head tag contains general information about the document.
  2. Title tag:
    • Whatever is written this tag comes up in the webpage, (or)Title of the webpage.
  3. Body tag:
    • Inside the html tag there was a body tag
    • This tag storing all information, and it contains image,links,tables,paragraph etc...
Attributes:
  • It is adding information to particular tag
  • Attributes will be always written inside starting tag
