Tags of HTML:

Types of tag:

  • There are two types of tags
    1. Paired tag:
      • These tag will be having opening tag as well as closing tag.
        • example: < h1>,< p>
    2. Unpaired tag:
      • These tag will be having only opening tag.
        • example: < br> ,< img>, < hr>
Types of Elements:Tags are also known as elements.
  1. Block level elements
    • Block level elements always start from new line.
  2. Inline elements
    • Inline elements never start from a new line.always start from previous element

Text formating tags in HTML:

  1. Bold tag < b >< /b >:
    • This tag is used to convert text in bold.
    • Another name of this tag < strong >< /strong >
      • Ex:< b >Webpage< /b

  2. Italic tag < i >< /i >:
    • This tag is used to convert text in Italic.
    • Another name of this tag (empasis)< em >< /em >
      • Ex:< i >webpage< /i >

  3. Underline tag < u >< /u >:
    • This tag is used to convert text in Underline.
    • Another name of this tag (inserted)< ins >< /ins >
      • Ex:< u >Webpage< /u

  4. Strike tag < strike >< /strike >:
    • This tag is used to horizontal line over the text.
    • Another name of this tag (del)< del >< /del >
      • Ex:< strike >Webpage< /strike

  5. Superscript tag < sup >< /sup >:
    • This tag is used to power of something.
      • Ex: o< sup > 2 < /sup >

  6. Subscipt tag < sub >< /sub >:
    • This tag is used to chemical formula.
      • Ex:< sub >Webpage< /sub >

    Tags

    1. Heading tag:< h1 >.....< h6 >
      • < h1 > < /h1 > tag is largest Heading.
      • < h6 > < /h6 > tag is smallest heading.
    2. Break tag: < br > < /br >
      • This tag is used to break line, moving to next line.
    3. Paragraph tag:< p >< /p >
      • This tag is used to write a paragraph.
    4. Comments:< ! -- -- >
      • Comments are not displayed in browser.
