List of HTML:

Ordered list < ol> < /ol>

  • Series of events that take place in some order
  • Inside the < ol> < /ol> used < li> < /li>

How to create order list:
< ol>
< li>Data< /li>
< li>Data< /li>
< li>Data< /li>
< /ol>

Attribute of < ol> < /ol>


  • type =" "
  • start =" "

2.Unordered list < ul> < /ul>

  • Series of events that take where order is not important.
  • Inside the < ul> < /ul> used < li> < /li>
How to create unordered list:
< ul>
< li>Data< /li>
< li>Data< /li>
< li>Data< /li>
< /ul>

Attribute of < ul> < /ul>

  • type ="circle"
  • type ="square"
  • type ="disc"
