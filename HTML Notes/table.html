Table tag of HTML:

  • This tag is a paired tag.It is used create a table on a webpage.
  • We need other tag for completing the creation of a table.
  • Inside table tag use < tr >< /tr > tag table row and < th >< /th > tag table heading and < td >< /td> tag table data.

Attribute of < table > tag.


< table bgcolor=" ", border=" "> < /table >

Rowspan/Colspan:

  • If a single cell accupy morethan one row morethan one column is know as rowspan and colspan.
  • These are the attributes given for either < th > or < td >tag specify the number of rows and columns. its accupies
